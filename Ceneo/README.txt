Aplikacja została stworzona przy użyciu Python 3.8, zatem sugerujemy aby uruchamiać ją w wersji 3.x.

Do uruchomienia aplikacji niezbędne jest posiadanie następujących bibliotek zewnętrznych:
- flask
- scrapy

Aby uruchomić aplikację, uruchom plik app.py, następnie w przeglądarce internetowej
połącz się z adresem "http://127.0.0.1:5000/" lub "http://localhost:5000/".
