Aplikacja została stworzona przy użyciu Python 3.8, należy uruchamiać ją w wersji 3.x 
(sugerujemy 3.8).

Do uruchomienia aplikacji niezbędne jest posiadanie następujących bibliotek zewnętrznych:
- flask
- scrapy

Aby uruchomić aplikację, uruchom plik app.py, następnie po wyświetleniu w terminalu komunikatu 
"Running on http://127.0.0.1:5000/", połącz się z podanym adresem za pomocą przeglądarki internetowej.
